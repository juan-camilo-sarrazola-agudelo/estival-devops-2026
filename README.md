# Festival DevOps Music Fest

Proyecto desarrollado como parte del curso DevOps y Contenedores (Docker) – SENA CTMA.

## Tecnologías utilizadas
- HTML5 y CSS3 (frontend)
- Python / Flask (backend)
- Docker y Dockerfile
- Docker Compose
- Git y GitHub para control de versiones

## Estructura del proyecto
- `frontend/`: landing page del festival.
- `backend/`: API construida con Flask.
- `docker-compose.yml`: orquestación de contenedores.

## Flujo de trabajo Git
Se utilizó Git Flow básico:
- `main`: rama principal estable.
- `feature-landing`: desarrollo de la landing page.
- `feature-backend`: desarrollo de la API Flask.
- `feature-artistas`, `feature-tickets`, `feature-contacto`: funcionalidades adicionales (reto avanzado).

Cada funcionalidad se desarrolló en su propia rama y se fusionó (merge) a `main` tras completarse.
